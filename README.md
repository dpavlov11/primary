# Персональный репозиторий проектов

Здесь хранятся мои учебные и рабочие материалы.

## Структура репозитория:
- **education/** — папка с учебными кейс-задачами.
  - **Case_2/** — программа для расчета факториала.
  - **Case_3/** — игра "Угадай число".
- **work/** — папка для рабочих проектов и черновиков.

## Окружение:
- Язык: Python 3.10+
- Инструменты: Git, PyCharm
