# Вычисление факториала (Python)
Программа на Python для вычисления факториала целого числа. Проект выполнен в рамках Учебной практики Кейс-задачи №2.

## Особенности
- **Обработка ошибок**: Программа проверяет ввод на корректность (только целые положительные числа).
- **Оптимизация**: Используется встроенная библиотека `math` для быстрых вычислений на уровне C.
- **Поддержка больших чисел**: Снято ограничение на количество знаков при выводе гигантских результатов с помощью `sys.set_int_max_str_digits`.

## Требования
- Python 3.10.7 или выше (из-за настроек лимита цифр вывода).

## Как запустить
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Перейдите в папку проекта:
   ```bash
   cd education
   ```
3. Запустите скрипт:
   ```bash
   python task_2.py
   ```
