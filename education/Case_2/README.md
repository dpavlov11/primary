# Кейс-задача №2: Вычисление факториала

Программа на Python для расчета факториала положительного целого числа.

### Функционал:
- **Валидация**: Проверка на ввод текста и отрицательных чисел.
- **Оптимизация**: Использование библиотеки `math` для вычислений на уровне C.
- **Большие данные**: Поддержка вывода чисел с неограниченным количеством знаков (`sys.set_int_max_str_digits`).

### Запуск:
```bash
python task_2.py
```
