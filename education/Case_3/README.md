# Образовательные проекты (Python)

Репозиторий содержит решения кейс-задач.

## 1. Вычисление факториала (task_2.py)
Программа для быстрого расчета факториала больших чисел.
- **Особенности**: Оптимизация `math.factorial`, обработка ошибок ввода.

## 2. Игра "Угадай число" (guess_number.py)
Интерактивная игра с ограничением попыток.
- **Особенности**: Валидация диапазона, система подсказок, ограничение по количеству попыток (10).

## Как запустить
```bash
python task_2.py       # Запуск факториала
python guess_number.py # Запуск игры
```
