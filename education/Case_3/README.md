# Кейс-задача №3: Игра "Угадай число"

Консольная игра, в которой пользователь должен угадать число от 1 до 100 за ограниченное число попыток.

### Особенности:
- **Интерактивность**: Подсказки "Слишком большое" или "Слишком маленькое".
- **Игровая логика**: Лимит в 10 попыток и умная подсказка (диапазон), когда остается 3 попытки.
- **Безопасность**: Обработка некорректного ввода (буквы, пустые строки).

### Запуск:
```bash
python task_3.py
```
